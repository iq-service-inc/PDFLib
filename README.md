# DocWatermarkPDF

這個專案是用來測試是否能符合文件浮水印加壓需求之測試專案


## System Requirement

* .NET Framework 4.5 以上

## Project Info

* DocWatermarkDLL -- Lib
* DocWatermarkPDF -- 測試呼叫
